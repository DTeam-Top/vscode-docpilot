# Workflow

## Features Development

```text
/dev:start (new feature) or /dev:continue (unfinished feature)
    -> discussion & coding
    -> /code:clean
    -> /dev:save
    -> /reflect
```

## Code Refactoring

```text
/code:review
    -> /code:refactor
    -> /code:clean
    -> /dev:save
    -> /reflect
```
